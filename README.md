# Plataforma-Distribuida-de-Monitoramento-de-Sistemas
Sistema de monitoramento de máquinas baseado em arquitetura distribuída cliente-servidor, com coleta, armazenamento e visualização de métricas em tempo real.

# Agente Coletor

Coleta métricas de CPU, memória e disco
na máquina monitorada e envia
periodicamente ao servidor.

# Servidor Central

Recebe os dados via API REST e os
armazena em banco de dados para
consulta e análise.

# Interface Web

Exibe indicadores e gráficos do
comportamento dos recursos ao longo
do tempo.
